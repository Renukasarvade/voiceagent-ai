# 🎙️ VoiceAgent AI

A full-stack AI agent system that converts user input (text/voice) into actionable tasks like code generation, file creation, and summarization.

---

## 🚀 Live Demo

 🚀 **Live Demo Available — Try it here:**  https://voiceagent-ai.onrender.com

---

## 🧠 Features

- Intent Detection using LLM
- Code Generation
- File Creation & Management
- Summarization
- Chat-based interaction

---

## ⚙️ Tech Stack

- Frontend: React (Vite)
- Backend: FastAPI
- AI Models: Groq / OpenRouter (LLMs)
- Deployment: Render

---

## 🏗️ Architecture

User Input → Frontend → FastAPI Backend → LLM (Intent Detection) → Action Execution → Response

---

## ⚡ Agent Flow

1. User gives command  
2. System detects intent (write_code, create_file, etc.)  
3. Backend executes action  
4. Result returned to user  

---

## 🚧 Limitations

- Uses predefined intents (not fully autonomous agent)  
- No long-term memory  
- Limited multi-step reasoning  

---

## 🔮 Future Improvements

- Multi-step reasoning agents  
- Memory integration  
- Tool chaining
